# Spring Boot Login App

A simple Login and Registration web application built using **Spring Boot, Thymeleaf, and MySQL**.

---

## Features

- User Registration  
- User Login  
- Session-based Authentication  
- Logout Functionality  
- MySQL Database Integration  

---

## Technologies Used

- Java  
- Spring Boot  
- Thymeleaf  
- MySQL  
- Maven  

---
